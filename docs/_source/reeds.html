undersampling_occurence_potential_threshold_distribution_based This function is estimating the pot_tresh for all states by testing if around the minimal energy 90% of the data is located in a threshold of max_distance_kJ. The mean and std of the density region will result in pot_tresh = mean+3*std This function is cheap in performance.

reeds.tmp_file.undersampling_occurence_potential_threshold_distribution_based(ene_traj_csvs: List[pandas.core.frame.DataFrame], sampling_fraction_treshold: float = 0.9)[source]¶

This function is estimating the pot_tresh for all states by testing if around the minimal energy 90% of the data is located in a threshold of max_distance_kJ. The mean and std of the density region will result in pot_tresh = mean+3*std This function is cheap in performance.

Parameters

  • ene_traj_csvs (List[pd.Dataframe]) – a list of pandas dataframes containing the energy data of each state eX

  • sampling_fraction_treshold (float, optional) – the sampling threshold for detecting undersampling (default: 0.9).

Returns

list of individual identified potential thresholds

Return type

List[float]
